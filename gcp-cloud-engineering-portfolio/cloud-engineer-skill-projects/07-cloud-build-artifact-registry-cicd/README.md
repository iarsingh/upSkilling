# Project 07: Cloud Build and Artifact Registry CI/CD

## Skill
CI/CD pipelines, image builds, test automation, artifact promotion, release gates, and deployment automation.

## Build
Create a Cloud Build pipeline that tests, builds, scans, stores, and deploys a container.

## Deliverables
- `cloudbuild.yaml`.
- Unit test stage.
- Docker build and push to Artifact Registry.
- Deployment to Cloud Run or GKE.
- Manual approval gate for production.

## GCP Services
Cloud Build, Artifact Registry, Cloud Deploy, Cloud Run, GKE.

## Proof Of Work
- Successful pipeline run screenshot.
- Image digest.
- Deployment revision or Kubernetes rollout result.
