# Project 08: GitOps with Argo CD on GKE

## Skill
GitOps, Kubernetes deployment promotion, environment overlays, reconciliation, rollback, and release traceability.

## Build
Install Argo CD on GKE and deploy applications from Git using Kustomize or Helm.

## Deliverables
- Argo CD installation manifests.
- App-of-apps structure.
- `dev`, `stage`, and `prod` overlays.
- Rollback and promotion runbook.

## GCP Services
GKE, Artifact Registry, Cloud DNS, Secret Manager.

## Proof Of Work
- Argo CD sync screenshot.
- Git commit triggering deployment.
- Rollback evidence.
