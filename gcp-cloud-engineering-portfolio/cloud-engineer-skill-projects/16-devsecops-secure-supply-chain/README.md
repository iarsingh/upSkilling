# Project 16: Secure Software Supply Chain

## Skill
SAST, dependency scanning, IaC scanning, container scanning, provenance, signing, and deployment gating.

## Build
Create a secure CI/CD pipeline that prevents untrusted artifacts from reaching production.

## Deliverables
- CI/CD pipeline with test, scan, build, sign, and deploy stages.
- Dependency scan.
- IaC scan.
- Container vulnerability scan.
- Provenance or attestation notes.

## GCP Services
Cloud Build, Artifact Registry, Binary Authorization, Cloud Deploy, GKE.

## Proof Of Work
- Pipeline blocked by a failed scan.
- Pipeline passes after remediation.
- Signed image or attestation evidence.
