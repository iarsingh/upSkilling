# Project 13: Secrets and Key Management

## Skill
Secret lifecycle, encryption, key rotation, service account security, workload identity, and least privilege.

## Build
Create a secure secrets management pattern for apps on Cloud Run and GKE.

## Deliverables
- Secret Manager setup.
- KMS key and rotation policy.
- Workload Identity access from GKE.
- Cloud Run secret injection example.
- Secret rotation runbook.

## GCP Services
Secret Manager, Cloud KMS, IAM, GKE, Cloud Run.

## Proof Of Work
- App reads secret without storing it in source control.
- IAM policy evidence.
- Rotation test notes.
