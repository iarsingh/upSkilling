## Interview Questions and Answers

### 1. What is BigQuery?
BigQuery is GCP's serverless data warehouse for analytics at scale.

### 2. What is partitioning?
Partitioning splits a table by date/time or integer range so queries scan less data.

### 3. What is clustering?
Clustering organizes data by column values within partitions to improve filtering performance.

### 4. What is Pub/Sub?
Pub/Sub is a messaging service for event ingestion and asynchronous communication.

### 5. What is Dataflow?
Dataflow is GCP's managed Apache Beam service for batch and streaming pipelines.
