# Project 00: FastAPI Basics Health API

## Skill
FastAPI routing, health checks, OpenAPI docs, environment variables, and service metadata.

## Run

```sh
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Open:

- `http://127.0.0.1:8000/`
- `http://127.0.0.1:8000/healthz`
- `http://127.0.0.1:8000/docs`
