# Project 03: Terraform Modules for GCP Infrastructure

## Skill
Infrastructure as Code, reusable modules, remote state, workspaces/environments, review workflows, and drift detection.

## Build
Create reusable Terraform modules for GCP projects, networks, service accounts, storage buckets, and GKE.

## Deliverables
- `modules/` folder for reusable infrastructure.
- `envs/dev`, `envs/stage`, and `envs/prod` examples.
- Remote state design using GCS.
- Terraform validation checklist.

## GCP Services
Cloud Storage, IAM, VPC, GKE, Cloud Build.

## Proof Of Work
- `terraform fmt`, `terraform validate`, and `terraform plan` output.
- Module README files.
- Drift detection notes.
