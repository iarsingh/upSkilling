# Project 15: Policy as Code Guardrails

## Skill
Policy as Code, compliance automation, admission control, Terraform checks, and preventive controls.

## Build
Create guardrails that block risky infrastructure and Kubernetes changes.

## Deliverables
- OPA/Conftest policies for Terraform.
- Gatekeeper or Policy Controller constraints for GKE.
- Examples of allowed and denied changes.
- Developer-friendly policy failure messages.

## GCP Services
GKE Policy Controller, Cloud Build, Terraform, Cloud Asset Inventory.
