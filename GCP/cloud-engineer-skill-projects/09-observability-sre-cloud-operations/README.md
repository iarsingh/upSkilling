# Project 09: Observability and SRE on GCP

## Skill
Monitoring, logging, tracing, SLI/SLO design, alerting, dashboards, and reliability reviews.

## Build
Create observability for an API running on Cloud Run or GKE.

## Deliverables
- SLI/SLO document.
- Cloud Monitoring dashboard.
- Alert policies for latency, error rate, saturation, and availability.
- Structured logging and trace correlation.

## GCP Services
Cloud Monitoring, Cloud Logging, Cloud Trace, Error Reporting, Managed Service for Prometheus.
