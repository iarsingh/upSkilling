## Interview Questions and Answers

### 1. What is DNS?
DNS maps names like `example.com` to IP addresses.

### 2. What is TLS?
TLS encrypts traffic and verifies server identity using certificates.

### 3. What is a load balancer?
A load balancer distributes traffic across multiple backends and can perform health checks.

### 4. What is CIDR?
CIDR represents IP ranges, such as `10.0.0.0/24`.

### 5. How do you debug connectivity?
Check DNS, routing, firewall rules, service health, TLS certificates, and application logs.
