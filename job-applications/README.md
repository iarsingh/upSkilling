# Job Applications

This folder is the working kit for high-volume applications on LinkedIn and Naukri.

## Candidate Defaults

- Experience: 7 years
- Current CTC: 13 LPA
- Expected CTC: 19 LPA
- Notice period: 90 days
- Location: Any
- Work mode: Remote, hybrid, or onsite
- Daily target: Maximum, recommended as 40-70 targeted applications per day

## Files

- `application-profile.json`: Your structured job-search profile.
- `search-links.md`: LinkedIn and Naukri search links.
- `tracker.md`: Application tracker.
- `message-templates.md`: Recruiter, cover note, and follow-up messages.

## Recommended Daily Batch

1. LinkedIn Easy Apply: 20-30 applications.
2. Naukri quick apply: 20-40 applications.
3. Recruiter messages: 10-15 messages for best-fit roles.
4. Tracker update: every application, same day.
5. Follow-ups: after 3 business days.

## Best Fit Keywords

Prioritize roles mentioning at least 3 of:

GCP, Kubernetes, Docker, Terraform, Ansible, CI/CD, GitOps, ArgoCD, Jenkins, GitHub Actions, MLflow, Kubeflow, Vertex AI, Prometheus, Grafana, OpenTelemetry, Python, Linux, SRE, MLOps.

## Application Safety

Do not auto-submit applications blindly. Confirm company, role, CTC range, location, and resume selection before submitting.
