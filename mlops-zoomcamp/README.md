# mlops-zoomcamp

Local workspace for working through [DataTalksClub's MLOps Zoomcamp](https://github.com/DataTalksClub/mlops-zoomcamp) course.

## Contents

- `mlops-zoomcamp/` — a full clone of the official course repository (modules `01-intro` through `07-project`, cohorts, homework). It's a separate git repository nested inside this folder and is intentionally excluded from this repo's tracked history via `.gitignore`, so course material stays independently versioned against the upstream project.
- `.venv/` — local Python virtual environment for running course notebooks/scripts (not tracked).

This folder is a learning workspace, not a showcase project — see the [root README](../README.md) for the portfolio-facing projects in this repository.
