# rag-deployment-projects

Local sandbox for RAG (Retrieval-Augmented Generation) deployment experiments.

## Contents

- `python/pdf-rag-reader/` — an in-progress PDF-based RAG service scaffold (`app/` package). Source files are currently local-only/untracked (only compiled `__pycache__` artifacts remain in the working tree); the intended shape is an ingestion + retrieval + LLM-answering pipeline over PDF documents.
- `.venv/` — local Python virtual environment (not tracked).

This is a working experiment folder rather than a finished showcase project — see the [root README](../README.md) for the polished, portfolio-facing projects in this repository.
