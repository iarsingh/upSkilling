# GitHub Achievements Notes

GitHub profile achievements are earned from public activity such as merged pull requests, fast issue handling, accepted discussion answers, stars, sponsorships, and collaborative commits.

## Practical Paths

- Pull Shark: merge pull requests.
- YOLO: merge your own pull request without a review.
- Quickdraw: close an issue or pull request within five minutes of opening it.
- Pair Extraordinaire: merge a pull request that includes a co-authored commit.
- Galaxy Brain: get an answer accepted in GitHub Discussions.
- Starstruck: create a repository that receives stars.
- Public Sponsor: sponsor an open source maintainer through GitHub Sponsors.

## Portfolio-Friendly Habit

Use small, real pull requests for learning notes, CI fixes, project docs, and examples. This builds profile activity while keeping the repository useful.
