# youtube-content-automation

Content automation for YouTube: generating and uploading MLOps/DevOps/Kubernetes/AI-infrastructure videos.

## Contents

- [content-automation/](content-automation/README.md) — local video generator/uploader. Uses Ollama to draft titles, descriptions, tags, thumbnail text, scenes, and narration, with a built-in MLOps script fallback when Ollama is offline.
