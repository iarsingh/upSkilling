## Deploy Options

### GitHub Pages

1. Push the `web-resume` folder to GitHub.
2. In repository settings, enable GitHub Pages.
3. Set the publishing source to the branch/folder that contains this `index.html`.

### Netlify

1. Create a new Netlify site from Git.
2. Set the publish directory to `web-resume`.
3. Leave the build command empty.

### Vercel

1. Import the repository in Vercel.
2. Set the output directory to `web-resume`.
3. Leave the build command empty.

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080 --directory web-resume
```

Then visit `http://localhost:8080`.
