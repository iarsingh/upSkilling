Built a tool to convert a static resume file into a customizable web page.

Project: Resume Web Builder — Turning an Uploaded Resume Into a Web Resume
GitHub: https://github.com/iarsingh/upSkilling/tree/main/resume-web-builder

What it does:
- Static frontend that parses uploaded resume files
- Outputs a customizable web resume layout
- A reusable tool, not a one-off personal site

Why it matters:
Building the tool instead of just using it once means I can regenerate and restyle my resume presence anytime.
