When the right dataset didn't exist, I generated one.

Project: MLOps Model Monitoring and Drift Dataset
GitHub: https://github.com/iarsingh/upSkilling/tree/main/kaggle-mlops-datasets/model-monitoring-drift

What it does:
- Synthetic prediction events from a customer churn model deployed across multiple regions
- Built specifically for drift detection and production ML monitoring practice
- Packaged as a Kaggle-ready dataset with a data dictionary

Why it matters:
Practicing drift detection needs realistic prediction telemetry — public datasets for this are scarce, so I built one that mirrors a real churn-model deployment.

Have you ever generated a synthetic dataset because the real one wasn't available or shareable?
