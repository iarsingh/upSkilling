# Learning Resources

Use official docs first. They are better for interviews because answers sound precise and real.

## Linux

- Linux command practice: https://linuxjourney.com/
- Shell scripting guide: https://www.shellscript.sh/
- Filesystem hierarchy reference: https://refspecs.linuxfoundation.org/FHS_3.0/fhs/index.html

What to learn:

- `ls`, `cd`, `cp`, `mv`, `rm`, `find`, `grep`, `awk`, `sed`
- `chmod`, `chown`, users and groups
- `ps`, `top`, `df`, `du`, `free`, `netstat` or `ss`
- logs under `/var/log`

- CI pipeline for lint, test, build
